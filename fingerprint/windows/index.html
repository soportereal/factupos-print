Instalacion

  1. Descargue todos los archivos y mantenga la estructura de carpetas: 
    factupos-fingerprint-windows\
  instalar.bat
  reiniciar_servicio.bat
  src\FingerprintService.cs
  sdk\DPFPDevNET.dll
  sdk\DPFPShrNET.dll
  sdk\DPFPEngNET.dll
  sdk\DPFPVerNET.dll
  sdk\DPFPGuiNET.dll
  2. Conecte el lector U.are.U 4500 al USB
  3. Click derecho en instalar.bat → Ejecutar como administrador
  4. Abra Chrome: https://127.0.0.1:52181/status
  5. Acepte el certificado → debe ver {"ok":true,"connected":true}

No necesita Visual Studio. El compilador C# ya viene incluido en Windows 10/11.
